# Terminal:

## **Commands**:

- **ls** - list directory contents
- **pwd** - print working directory (show the path to current directory)
- **cd** - change directory
- **mkdir** - make directory
- **touch** - create empty file
- **rm** - remove
- **cat** - concatenate
- **file** - determine file type
- **cp** - copy
